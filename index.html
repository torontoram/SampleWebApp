Command Description Example Usage
docker run Run a command in a new container, often used to create and start a container from an image. docker run -it ubuntu /bin/bash
docker ps List running containers. Use -a to list all (running and stopped) containers. docker ps -a
docker build Build an image from a Dockerfile. docker build -t my-app:latest .
docker pull Pull an image or a repository from a registry (like Docker Hub). docker pull nginx:latest
docker images List local Docker images. docker images
docker stop Stop one or more running containers. docker stop container_id
docker rm Remove one or more containers. docker rm container_id
docker rmi Remove one or more images. docker rmi image_id
docker exec Execute a command in a running container. docker exec -it web-server /bin/sh
